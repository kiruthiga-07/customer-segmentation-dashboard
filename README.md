# Customer Segmentation Dashboard

This project is a Streamlit-based data science dashboard that performs customer segmentation using clustering algorithms.

## Algorithms Used
- K-Means Clustering
- Hierarchical Clustering
- DBSCAN
- PCA (Principal Component Analysis)

## Features
- Upload CSV dataset
- Select features for clustering
- Choose clustering algorithm
- Select number of clusters
- PCA visualization
- Cluster statistics
- Download clustered data

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib

## How to Run (Local)

pip install -r requirements.txt  
streamlit run app.py

## Streamlit Cloud

This project can be deployed using Streamlit Cloud by connecting this GitHub repository.
